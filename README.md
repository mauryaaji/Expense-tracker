# 💸 Expense Tracker PRO

A powerful GUI-based Expense Tracker built using Python Tkinter.

## 🚀 Features
- 🔐 Login & Signup system
- 💰 Add, Delete, Search Expenses
- 📊 Pie Chart & Bar Graph visualization
- 🧠 Smart Category Suggestions
- 📂 JSON-based data storage

## 🛠 Tech Stack
- Python
- Tkinter
- Matplotlib
- JSON
